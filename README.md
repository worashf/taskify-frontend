# taskify-frontend

> Task management application Built by ReactJS, Redux Toolkit. Users can create a kanban board for the project, add tasks for the project, update the tasks state and Assign a person.

## Built With

- ReactJS
- Redux Toolkit
- SCSS

# [Online demo](#)

## Getting Started

To get the copy of this project in your computer,clone this repository or just download it.

### Prerequisites

- Code editor like vs code
- node js installed

### Setup

- Download or clone the repository
- Open the folder in your IDE
- Install the dependencies
- Open the index.html in your browser and check it
- Add your addition
- Push to your github or deploy to cloud
